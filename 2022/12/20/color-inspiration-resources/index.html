web resources to get color palette inspiration

Posted on 2022-12-20 Edited on 2022-12-21 In tool usage

When you struggle on the colors visualizing your results, always good to get some inspiration from color repository.These websites might be useful.

Palette sets

Those websites provide fixed palette,you can also create as you like.

  • coolors
    This website provides massive combinations of colors and very flexible functions to create your own palette and gradient color scheme.
  • viz-palette
    This website directly exhibits the color palette your create in different plot formats like barplot, boxplot, scatterplot, etc.
  • colorbrewer
    This website projects the colors in a map composed of small polygons to intuitively show their distinguishability.
  • uigradients
    This website give you the view of multiple gradient color schemes, two terminal colors are given in each of them for your own use.
  • ggsci
    This is a R package, which provides fabulous color palettes prefered by a few top class scientific journals like JAMA, NEJM, etc.

Colors extraction from pictures

By uploading pictures, those tools can extract the colors in them.

  • PaletteGenerator
  • Adobe Color
